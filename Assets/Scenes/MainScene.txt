Unity 2D 횡스크롤 씬 구성 안내

1. 새 2D 씬을 생성하고 이름을 MainScene으로 지정합니다.
2. 플레이어 오브젝트:
   - 빈 게임 오브젝트 생성 후 이름을 Player로 지정
   - Sprite Renderer 추가 (임시 박스 또는 캐릭터 스프라이트 사용)
   - Rigidbody2D 추가, Body Type은 Dynamic
   - Collider2D 추가 (BoxCollider2D 또는 CircleCollider2D)
   - PlayerController 스크립트 추가
3. GameManager:
   - 빈 게임 오브젝트 생성 후 이름을 GameManager로 지정
   - GameManager 스크립트 추가
4. 플랫폼/지면:
   - 바닥 역할을 하는 Sprite/Tilemap 추가
   - Collider2D 부착
5. 카메라:
   - 메인 카메라를 플레이어를 따라다니도록 설정
   - 필요한 경우 Cinemachine 도입

참고: 이 텍스트는 Unity 에디터에서 씬을 구성할 때 참고용입니다.
